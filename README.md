Candy Distributor Sales Analytics

[Overview](Overview.png)


Project Overview
- This project is an interactive Power BI dashboard built to analyze sales performance, profitability, forecasting, and customer insights for a candy distribution business.

---

Business Objective
- To help stakeholders track sales performance, measure profitability, analyze customer behavior, and forecast future sales trends using interactive visual analytics.



Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling
- Data Visualization



Key Performance Indicators (KPIs)
- Year-on-Year Profit %
- Year-on-Year Sales %
- Year-on-Year Units Sold %



Dashboard Features

Overview Page
- Product Target Achievement %
- Division-wise Sales vs Target
- Region-wise Contribution %
- Profit Margin Analysis
- Top 10 Customers by Sales
- KPI Cards for business performance

Trend Analysis Page
- Top 10 Cities by Sales (Map)
- Sales Forecast (Next 6 Months)
- Weekday vs Weekend Sales Analysis
- Monthly vs Quarterly Sales Trends (Bookmarks)
- 3-Month Moving Average



Interactive Features
- Year slider filter
- Month selection slicer
- Division filter (Chocolate / Sugar / Others)
- Clear all filters button
- Page navigation (Overview ↔ Trend Analysis)
- Bookmark-based view switching



Key Insights
- Identified top-performing product divisions and customer segments.
- Sales performance varies significantly by region and product category.
- Forecasting helps anticipate demand for the next 6 months.
- Weekend vs weekday patterns show different purchasing behavior trends.
